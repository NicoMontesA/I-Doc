#Requisitos

El sistema desarrollado esta diseñado para Python 3, además se requieren de las siguientes bibliotecas:

PyQt5 : Bibliteca gráfica que permite la creación de las ventanas, para la interacción máquina-usuario
Reportlab: Biblioteca que genera el PDF, de acuerdo a los datos del paciente.

También se crearon los módulos:

Enfermedades: contiene la clases Jaqueca, Resfrií y Gastroenteritis
CreacionReceta: contiene la función generadora de la receta
